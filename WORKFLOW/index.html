üìã Workflow Tipico

Scenario 1: Nuova bozza di blog post

git preview                    # Vai su Local_writing
# scrivi il post...
git staging                    # Push su staging
# Visita https://manuelcolombo.github.io/preview per vedere l'anteprima

Scenario 2: Fix veloce in produzione

# fix il bug...

# Testa in locale...
